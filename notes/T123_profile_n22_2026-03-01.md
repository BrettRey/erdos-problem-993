# T1/T2/T3 Rescue Mechanism Profile (n=21-22 extension, 2026-03-01)

## Decomposition

The 3-term SCC identity: `b_{k-1}·Δ_k = T1 + T2 + T3` where:
- T1 = b_{k-1}·d_k (current LR minor, can be negative)
- T2 = b_k·d_{k-1} (memory from adjacent index)
- T3 = a_{k-1}·c_k (LC curvature amplification, always >= 0)

## Key findings

1. **Curvature alone always suffices** (T2 never needed) through n=22
2. Min ratio decreasing: 7.09 (n=21) -> 4.23 (n=22), but still comfortable
3. d_k < 0 never occurs at k=1 (only k >= 2)
4. The tightest tree has large IS polynomial coefficients, indicating a bushy structure
