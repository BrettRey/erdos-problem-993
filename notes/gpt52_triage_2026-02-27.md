## Instance 1: Extendability reformulation

**Verdict: Useful reformulation, no new proof content.**

The key identity is standard but nicely framed:

  a_k := E(|A(S)| | |S|=k) = (k+1) i_{k+1} / i_k

and mode ≤ ⌈μ⌉ ⟺ a_k < k+1 for all k ≥ ⌈μ⌉.

The Glauber dynamics "finish line" is the cleanest target statement: if k → a_k - k is strictly decreasing for trees, the conjecture follows. But the output honestly acknowledges this requires new input (hard-core measure on trees is repulsive, no FKG).

The MLR condition (i_k(T'-u)/i_k(T') nonincreasing in k) for leaf-attachment induction is interesting but unproved, and the output correctly notes it would normally follow from real-rootedness/strong Rayleigh, which trees don't have.

**Connection to Basit-Galvin:** Their "ordered LC" conjecture is exactly "a_k decreasing". Since LC failure ⟹ ordered LC failure (proof: LC failure at k means r_k > r_{k-1}, which forces a_k > a_{k-1}), ordered LC is FALSE for trees (fails at n=26). But a_k < k+1 for k ≥ ⌈μ⌉ is strictly weaker and may still hold universally.

**Actionable:** The a_k framing is equivalent to mode ≤ ⌈μ⌉, which we've already verified on 9.1M trees. No new computational test needed — this is a useful lens, not new data.

## Instance 3: Strengthened induction (P⋆)

**Verdict: Most actionable. Concrete testable invariant. Should be computationally validated immediately.**

### The P⋆ proposal:

For a rooted tree (T,r), write E = dp[r][0], I = dp[r][1]/x. Then S = E + xI = I(T).

Property P⋆(E, I; m) consists of three conditions:

1. **P1 (Prefix LC):** Both E and I are LC on indices 0..m+1.
2. **P2 (Prefix TP2):** e_{k+1}·i_k ≥ e_k·i_{k+1} for k = 0,...,m-1. (Ratio i_k/e_k nonincreasing up to m.)
3. **P3 (Tail domination):** e_k ≥ i_{k-1} for all k ≥ m.

where m ≈ ⌈n/3⌉ (or ⌈μ⌉).

**Why this is well-designed:**
- Two-state (E, I), matching the DP's two-state structure
- Windowed (prefix only), compatible with known tail LC failures
- P1+P2 control the sum E+xI near the mode (TP2 + LC → mixture has single peak)
- P3 blocks secondary bumps in the tail
- Product preservation (P1+P2 should survive convolution via Gross et al.'s synchronisation theory)
- Does NOT imply global LC (so compatible with n=26 failures)

**Important caveat:** P⋆ depends on the rooting. For the induction to work, it must hold at EVERY subtree during the DP, not just at the top level.

**"Hu et al." reference is REAL** (corrected in Round 2 triage: H. Hu, D.G.L. Wang, F. Zhao, T.Y. Zhao, arXiv:1507.08430). Round 1 incorrectly called it fabricated. The synchronised sequences concept is from Gross-Mansour-Tucker-Wang; Hu et al. is the follow-up on *partial* synchronicity.

### The key computational test (HIGHEST PRIORITY):
For each tree T with n ≤ 20 (or n ≤ 23 for d_leaf ≤ 1):
- Root at each vertex r
- Compute E = dp[r][0], I = dp[r][1]/x
- Set m = ⌈n/3⌉ (or mode(I(T)))
- Check P2: is i_k/e_k nonincreasing for k = 0,...,m-1?
- Check P3: is e_k ≥ i_{k-1} for all k ≥ m?
- If any rooting passes, record it; if ALL rootings fail, record as failure

This is a concrete experiment that can validate or kill P⋆ in one run.

### SCAN RESULT (2026-02-27):
**9,114,283 trees n≤22, EXISTENTIAL, 0 failures.** For every tree, there EXISTS
at least one rooting where both P2 and P3 hold.

**CORRECTION:** The original scan had a bug — it short-circuited on the first passing
rooting and reported "pass." The claim that P⋆ holds at ALL rootings was WRONG.
Detailed per-rooting analysis shows:
- P3 fails at ~60% of rootings (binding constraint)
- P2 fails at ~15-20% of rootings
- Every P⋆ failure is a P3 failure (P3 is strictly harder)
- Example: P_5 rooted at center or endpoints fails P3; passes only at interior degree-2 vertices

The existential result (∃ good rooting for every tree) is still significant but
changes the proof strategy: we cannot prove P⋆ by induction at arbitrary rootings.
